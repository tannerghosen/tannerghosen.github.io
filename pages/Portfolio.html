Wasspord

GitHub Repo

This is a password management program made using WinForms in C# & .NET. The program saves account info added to this program in a .wasspord file with a user's name of choice. Attached to these files can be a password to unlock the file for future use. All passwords, including the one to unlock the program, are encrypted into the file so that it cannot be easily decrypted without the program. As bonus features, Wasspord comes with a password generator, password validator, miscellaneous settings, and autosaving functionality.

Password Generator API

GitHub Repo

This is a Password Generator API created with Express.js, Node.js and JavaScript. It can generate a randomly generated password of either a pre-determined or custom length, and sends back a JSON response containing it. Also includes a form of validation against a basic regex as an additional feature.
